# План: Экран мастера с базой данных

## Что будет сделано

1. **База данных Supabase** (бесплатно) — хранит все записи от всех клиентов
2. **API-эндпоинт `/api/book`** — при подтверждении записи клиентом сохраняет в базу + уведомляет мастера в Telegram
3. **API-эндпоинт `/api/bookings`** — отдаёт список записей (только для мастера)
4. **Вкладка «Расписание» в Mini App** — видна только мастеру (по Telegram ID)

---

## Шаг 1: Создать аккаунт Supabase ✅

1. Зайти на [supabase.com](https://supabase.com)
2. Нажать **Start your project** → войти через GitHub
3. Нажать **New project**
4. Заполнить:
   - **Name:** `beauty-catalog`
   - **Database Password:** придумать пароль (запомнить)
   - **Region:** Frankfurt (ближайший)
5. Нажать **Create new project** — ждать ~1 минуту

---

## Шаг 2: Создать таблицу и получить ключи

- Создать таблицу `bookings` в Supabase
- Скопировать `Project URL` и `anon key` из настроек
- Добавить их в переменные окружения Vercel

---

## Шаг 3: API-эндпоинты на Vercel

- `POST /api/book` — принимает данные записи, сохраняет в Supabase, шлёт уведомление мастеру в Telegram
- `GET /api/bookings` — возвращает все записи (защищён по Telegram ID мастера)

---

## Шаг 4: Изменения в Mini App

- При подтверждении записи → отправлять данные на `/api/book`
- Добавить 4-ю вкладку «Расписание» (видна только мастеру)
- Экран расписания: список записей с фильтром по дате, статус (активна / отменена)
