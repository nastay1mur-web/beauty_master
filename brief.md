# Бриф: Beauty Catalog TMA v1.0

> Telegram Mini App для соло-бьюти-мастера.
> Клиент: открыл бота → увидел услуги → нажал «Записаться» → готово.
> Не сайт. Приложение.

---

## Навигация

```
Шторка снизу (Bottom Tab Bar) — всегда видна:

[ 💅 Услуги ]  [ 📅 Мои записи ]  [ 👤 О мастере ]
```

Три таба. Ничего лишнего.

---

## Экраны

### ЭКРАН 1 — Каталог услуг (стартовый)

> Первое, что видит пользователь. Открывается автоматически при запуске TMA.

---

**Верхняя часть (шапка, не скроллится):**

```
┌─────────────────────────────────────┐
│  [Фото мастера 48px круг]  Катя     │
│  Мастер маникюра · ★ 4.9           │
└─────────────────────────────────────┘
```

- Аватар мастера — круглый, 48px
- Имя мастера — полужирный
- Специализация + рейтинг — серый текст, мелко

---

**Таб-категории (горизонтальный скролл, не скроллится вместе с контентом):**

```
[ Все ]  [ Маникюр ]  [ Педикюр ]  [ Лэши ]  [ Брови ]
```

- Пилюли / чипсы, горизонтальный скролл
- Активная вкладка — акцентный цвет (брендовый)
- Минимальная высота таба: 36px, touch-область 44px

---

**Список услуг (вертикальный скролл):**

Каждая карточка:

```
┌─────────────────────────────────────┐
│                                     │
│         [ФОТО РАБОТЫ]               │
│         полная ширина, 200px высота │
│                                     │
│  Маникюр с гель-лаком          [→]  │
│  ⏱ 90 мин          от 1 800 ₽       │
│                                     │
│  [    Записаться    ]               │
└─────────────────────────────────────┘
```

**Элементы карточки:**
- Фото (полная ширина, соотношение 16:9 или 4:3, без отступов по бокам)
- Название услуги — полужирный, 17px
- Значок ⏱ + длительность — серый, 14px
- Цена («от 1 800 ₽») — полужирный, 16px
- Стрелка → (тап на карточку = открывает детали)
- Кнопка «Записаться» — высота 48px, полная ширина карточки минус отступы

**Состояния карточки:**
- Скелетон при загрузке (серые прямоугольники на месте фото и текста)
- Если услуга недоступна — карточка серая, кнопка «Нет свободных слотов» (неактивна)

---

**Действия:**
| Элемент | Действие |
|---|---|
| Тап на категорию | Фильтрует список, остаётся на экране |
| Тап на карточку (не кнопку) | → Экран 2: Детали услуги |
| Кнопка «Записаться» | → Экран 2: Детали услуги |

### ЭКРАН 3 — Выбор даты и времени

> Выбрать день → выбрать слот. Два шага на одном экране.

---

**Telegram BackButton** → Экран 2: Детали услуги

---

**Верхняя часть (не скроллится):**

```
┌─────────────────────────────────────┐
│  Маникюр с гель-лаком               │  ← мини-напоминание что выбрали
│  Выберите дату                      │
│                                     │
│  [Вт]  [Ср]  [Чт]  [Пт]  [Сб] → │
│  [ 18] [19]  [20]  [21]  [22]  →   │
│         ↑ сегодня                   │
└─────────────────────────────────────┘
```

**Горизонтальный свайп дней (30 дней вперёд):**
- Каждый день — прямоугольник 52×64px
- Содержимое: день недели (Вт) мелко + число (18) крупно
- Состояния:
  - Есть слоты → белый/светлый фон, активный
  - Нет слотов → серый, неактивный (не тапается)
  - Выбранный → акцентный цвет фон, белый текст

---

**Нижняя часть — слоты (появляются после выбора дня):**

```
┌─────────────────────────────────────┐
│  Доступное время · среда, 19 марта  │
│                                     │
│  [ 10:00 ]  [ 11:30 ]  [ 13:00 ]   │
│  [ 14:30 ]  [ 16:00 ]  [  ——  ]    │  ← занят
│  [ 17:30 ]  [ 19:00 ]              │
│                                     │
└─────────────────────────────────────┘

[  Подтвердить — ср, 19 марта, 14:30  ]  ← MainButton (активна после выбора слота)
```

**Кнопки слотов:**
- Размер: минимум 44×44px, лучше 48px высота
- Свободный: белый/светлый фон, текст времени
- Занятый: серый фон, «——», неактивен (не тапается)
- Выбранный: акцентный цвет

**Telegram MainButton:**
- До выбора слота: скрыта или текст «Выберите время» (неактивна)
- После выбора: «Подтвердить — ср, 19 марта, 14:30» + активна

**Действия:**
| Элемент | Действие |
|---|---|
| Тап на день | Выделяет день, показывает слоты под ним |
| Тап на слот | Выделяет слот, активирует MainButton |
| MainButton | → Экран 4: Подтверждение |
| BackButton (←) | → Экран 2 (выбор слота не теряется) |

## Технические требования интерфейса

### Размеры и зоны

| Элемент | Минимальный размер |
|---|---|
| Любая кнопка | 48px высота |
| Touch-область | 44×44px |
| Фото карточки услуги | 100% ширина, 200px высота |
| Аватар мастера в шапке | 48px диаметр |
| Слот времени | 48px высота, 33% ширины |
| Таб категории | 36px высота, padding 12px |
| Карточка записи (Э6) | мин. 80px высота |

### Цвета (только Telegram CSS-переменные)

```css
/* Фон */          var(--tg-theme-bg-color)
/* Текст */        var(--tg-theme-text-color)
/* Серый текст */  var(--tg-theme-hint-color)
/* Фон карточек */ var(--tg-theme-secondary-bg-color)
/* Кнопки */       var(--tg-theme-button-color)
/* Текст кнопок */ var(--tg-theme-button-text-color)
/* Ссылки */       var(--tg-theme-link-color)

/* Единственный брендовый цвет: */
--brand-accent: #C9A96E;  /* нежно-золотой, бьюти-тон */
/* Использовать только для: активный таб, выбранный слот, выбранный день */
```

### Haptic Feedback — обязательно

| Событие | Вибрация |
|---|---|
| Тап на категорию | `selectionChanged()` |
| Тап на слот времени | `selectionChanged()` |
| Нажатие «Записаться» | `impactOccurred('medium')` |
| Экран успеха открылся | `notificationOccurred('success')` |
| Ошибка (нет слотов, сбой) | `notificationOccurred('error')` |
| Нажатие «Отменить запись» | `impactOccurred('rigid')` |

### Telegram API — обязательные вызовы

```javascript
// При старте
WebApp.ready()
WebApp.expand()

// На Э2, Э3, Э4 — показывать BackButton
WebApp.BackButton.show()
WebApp.BackButton.onClick(() => navigateBack())

// На Э1, Э6, Э7 — скрывать BackButton
WebApp.BackButton.hide()

// MainButton — настраивать контекстно
WebApp.MainButton.setText('Записаться')
WebApp.MainButton.show()

// Имя клиента — из данных Telegram
const { first_name, last_name } = WebApp.initDataUnsafe.user

// Подтверждение отмены записи
WebApp.showConfirm('Отменить запись?', (ok) => { if (ok) cancelBooking() })

// Ошибки
WebApp.showAlert('Не удалось записаться. Попробуйте ещё раз.')
```

### Состояния загрузки

- При первом открытии: скелетон (серые прямоугольники вместо карточек)
- При загрузке слотов: скелетон сетки слотов
- При отправке записи: `MainButton.showProgress(true)`
- При ошибке сети: баннер «Нет соединения» + кнопка «Повторить»

## Чего НЕ будет в версии 1.0

### Функции — отложены

| Что | Почему отложено |
|---|---|
| Онлайн-оплата (Telegram Payments / ЮКасса) | Требует согласования с провайдером (1-2 нед). Добавить в v1.1 после валидации флоу |
| Система отзывов и рейтингов | Нужна накопленная база записей. Запустить после 50+ записей |
| Акции и промокоды | Усложняет MVP. Итерация 2 |
| Программа лояльности (баллы, скидки) | Итерация 2 |
| Фильтры каталога (по цене, длительности) | У соло-мастера max 15-20 услуг, фильтры избыточны |
| Поиск по услугам | Категорий достаточно для навигации |
| Профиль клиента с редактированием | Telegram даёт имя, телефон — через `requestContact()` |
| Несколько мастеров | v1 — для соло. Мультимастер — v2 (другая архитектура) |
| Список ожидания (waitlist) | После v1, когда будет реальная загруженность |
| Повторяющаяся запись («каждый четверг») | После v1 |
| Push-рассылки от мастера (анонсы, фото) | Бот умеет, но UI для этого — v2 |

### Экраны — не в v1

| Экран | Причина |
|---|---|
| Экран акций / офферов | Нет акций в MVP |
| Страница отзывов | Нет отзывов в MVP |
| Онбординг / tour | Интерфейс достаточно прост |
| Настройки уведомлений | Базовые напоминания автоматически |
| Поиск | Не нужен при ≤ 20 услугах |
